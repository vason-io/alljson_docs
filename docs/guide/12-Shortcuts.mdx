# Shortcuts

**Alt+1** - focus on the `left editor`. [Text representation of JSON](/guide/visualization/text-representation-of-json)

**Alt+2** - focus on the `right editor`. [Text representation of JSON](/guide/visualization/text-representation-of-json)

**Alt+Q** - open the `query` window to analyze JSON in the left panel. [Jsonata queries](/guide/query-system/jsonata-queries)

**Alt+X** - open the `Xpath/XSLT` window to analyze JSON in the left panel. [XPath queries](/guide/query-system/xpath-queries) and [XSLT transformations](/guide/query-system/xslt-transformations)

**Alt+F** - open the `Filter` window to analyze JSON in the left panel. [Regular expressions filter and search](/guide/query-system/regular-expressions)

**Alt+T** - open the `HTTP` window (`HTTP requests` tab). [HTTP requests](/guide/http-client-and-automation/http-requests)

**Alt+V** - open the `HTTP` window (`HTTP variables` tab). [HTTP variables](/guide/http-client-and-automation/http-variables)

**Alt+P** - open the `HTTP` window (`HTTP script` tab). [Script](/guide/http-client-and-automation/script)

**Alt+Y** - open the `Python` window. [Python](/guide/python)

**Alt+C** - open the `CSV to JSON` window. [CSV to JSON](/guide/csv-to-json)

**Alt+P** - open the `HTTP` window (`HTTP script` tab). [Script](/guide/http-client-and-automation/script)

**Alt+S** - open the `Notes` window (this window serves as a clipboard or scratches texts). [Notes](/guide/notes)

**Alt+W** - open the `Workspaces` window. [Caching](/guide/caching)

To quickly access the previous window, just repeat the same shortcut.

**Ctrl+S** - save content in the browser cache and create a workspace automatically. [Caching](/guide/caching)

Shortcuts useful in panel editors (`left editor` and `right editor`) and `HTTP`, `query` or `Notes` windows:

**Ctrl+D** - duplicate text in editors

**Ctrl+Y** - delete line in editors

**Ctrl+B** - go to brackets in editors

**Ctrl+Shift+B** - select to brackets in editors

**Ctrl+F2** - toggle bookmark

**F2** - go to next bookmark
