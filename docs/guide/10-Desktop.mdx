# Desktop

https://alljson.com/open-desktop

Alljson is available for download as a desktop application.

In desktop environment, Alljson can execute HTTP requests directly without a CORS proxy.

Premium subscription is still required in order to execute HTTP requests.

Please, see [registration and subscription to the premium plan](/guide/http-client-and-automation/intro-http-client#registration-in-the-alljson-application)
