## Table view

[JSON data in a table form (https://alljson.com/open-table)](https://alljson.com/open-table)

The `Table` window allows you to view JSON data in a table form.

import { ThemedImg } from '@site/src/components/ThemedImg';

<ThemedImg alt="Table view" name="Table_view" />

1. Choose the `Table` element in `Extra view` dropdown on either the left or right side to open the `Table` window.

2. Subsequent levels of nesting open in new windows when clicking on the three dots enclosed in curly braces `{...}`
