# Shortcuts

**Alt+1** - focus on the `left editor`. [Text representation of JSON](/documentation/Visualization/Text%20representation%20of%20json)

**Alt+2** - focus on the `right editor`. [Text representation of JSON](/documentation/Visualization/Text%20representation%20of%20json)

**Alt+Q** - open the `query` window to analyze JSON in the left panel. [Jsonata queries](/documentation/Query%20system/Jsonata%20queries)

**Alt+X** - open the `Xpath/XSLT` window to analyze JSON in the left panel. [XPath queries](/documentation/Query%20system/XPath%20queries) and [XSLT transformations](/documentation/Query%20system/XSLT%20transformations)

**Alt+F** - open the `Filter` window to analyze JSON in the left panel. [Regular expressions filter and search](/documentation/Query%20system/Regular%20expressions)

**Alt+T** - open the `HTTP` window (`HTTP requests` tab). [HTTP requests](/documentation/HTTP%20client%20and%20automation/HTTP%20requests)

**Alt+V** - open the `HTTP` window (`HTTP variables` tab). [HTTP variables](/documentation/HTTP%20client%20and%20automation/HTTP%20variables)

**Alt+P** - open the `HTTP` window (`HTTP script` tab). [Script](/documentation/HTTP%20client%20and%20automation/Script)

**Alt+Y** - open the `Python` window. [Python](/documentation/Python)

**Alt+C** - open the `CSV to JSON` window. [CSV](/documentation/Csv)

**Alt+P** - open the `HTTP` window (`HTTP script` tab). [Script](/documentation/HTTP%20client%20and%20automation/Script)

**Alt+S** - open the `Notes` window (this window serves as a clipboard or scratches texts). [Notes](/documentation/Notes)

**Alt+W** - open the `Workspaces` window. [Caching](/documentation/Caching)

To quickly access the previous window, just repeat the same shortcut.

**Ctrl+S** - save content in the browser cache and create a workspace automatically. [Caching](/documentation/Caching)

Shortcuts useful in panel editors (`left editor` and `right editor`) and `HTTP`, `query` or `Notes` windows:

**Ctrl+D** - duplicate text in editors

**Ctrl+Y** - delete line in editors

**Ctrl+B** - go to brackets in editors

**Ctrl+Shift+B** - select to brackets in editors

**Ctrl+F2** - toggle bookmark

**F2** - go to next bookmark

**F1** - list of all editorÂ´s commands
