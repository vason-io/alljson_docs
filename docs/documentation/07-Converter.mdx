# Converter

Vason позволяет конвертировать текст из одного формата в другой.

На данный момент поддерживается возможность конвертации текста между следующими форматами:
**JSON**, **XML**, **TOML**, **YAML**

2. Выберите один из вариантов конвертирования
3. Например, можно сконвертировать JSON в XML и наоборот
4. JSON в TOML и наоборот
5. JSON в YAML и наоборот
