# Converter

[vason.io/open-converter](https://vason.io/open-converter)

Vason allows converting text from one format to another.

Currently, the ability to convert text between the following formats is supported:
**JSON**, **XML**, **TOML**, **YAML**

import { ThemedImg } from '@site/src/components/ThemedImg';

1. Open the converter window
2. Choose one of the conversion options
3. For example, you can convert JSON to XML and vice versa
4. JSON to TOML and vice versa
5. JSON to YAML and vice versa.
