# Desktop

Vason is available for download as a desktop application.

In desktop environment, vason can execute HTTP requests directly without a CORS proxy.

Premium subscription is still required in order to execute HTTP requests.

Please, see [registration and subscription to the premium plan](HTTP%20client%20and%20automation/Intro#registration-in-the-vason-application)
