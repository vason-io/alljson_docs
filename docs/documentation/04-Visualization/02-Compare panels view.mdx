# Compare panels view

[Compare texts in JSON format (https://vason.io/open-compare)](https://vason.io/open-compare)

The `Compare` window enables you to compare the content from two JSON panels.

import { ThemedImg } from '@site/src/components/ThemedImg';

<ThemedImg alt="Compare texts in JSON format" name="Compare_panels_view" />

1. Click the compare button to open the `Compare` window.

2. You can navigate through the differences by using prev or next buttons.

3. You can achieve cursor position synchronization between the left/right JSON panel and the corresponding left/right "Compare" window by following these steps:
   - Click on a specific location within the left side "Compare" window.
   - Next, click the ‚óÅ sync left button.
   - This action will automatically navigate you to the corresponding position in the original JSON on the left JSON panel.
