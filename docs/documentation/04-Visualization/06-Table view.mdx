# Table view

The `Table` window allows you to view JSON data in a table form.

import ThemedImage from '@theme/ThemedImage';

<ThemedImage

1. Choose the `Table` element in `Extra view` dropdown on either the left or right side to open the `Table` window.

2. Последующие уровни вложенности открываются в новых окнах по клику на три точки обрамлённые фигурными скобками `{...}`
