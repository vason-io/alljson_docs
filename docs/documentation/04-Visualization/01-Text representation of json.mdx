# Text representation of JSON


Правая и левая панели содержат текстовые редакторы для репрезентации Json в текстовом виде.

Here you can edit and analyze JSON in the text form.

1. Мини-панель слева от текстового редактора содержит следующие кнопки-команды:
    - undo changes
    - redo changes
    - select all
    - copy selected text (if some text was selected) or copy whole text to clipboard
    - paste text from clipboard
    - clear whole content and paste text from clipboard
    - wrap/unwrap text
    - prettify JSON
    - cut whole or selected text and copy to clipboard
    - clear selected or whole text

2. Сверху расположена панель навигации по структуре Json (aka breadcrumbs).
    Панель навигация всегда синхронизована с положением курсора в тексте.
    Также с помощью панели навигации можно перемещаться  по тексту выбирая тот или иной ключ в панели навигации.

3. По клавише F1 можно увидеть список всех команд (и клавиш вызова(shortcuts)) редактора

4. При нажатии на шеврон рядом с тем или иным ключом можно сворачивать и разворачивать элементы структуры

5. Кнопка "Prettify" позволяет prettify JSON

6. Также, например, одной из удобных возможностей редакторов, является работа с закладками:
    комбинацию клавиш Control+F2 можно использовать для переключения закладок. F2 - для перехода к следующей закладке
