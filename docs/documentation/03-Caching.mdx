# Caching

### Templates and Workspaces

Для ознакомления с примерами работы Vazon можно загрузить один из предустановленных Templates.

Также можно сохранять содержимое редакторов или запросы  в памяти браузера, и работать с Workspaces.

В памяти браузера кэшируется только выбранный контент.

### Caching and Saving

To save your documents (JSONs, HTTP requests, variables, script, and queries) in the browser's cache, simply select the caching checkboxes.

This will ensure that your texts or queries are consistently updated in the browser cache (IndexedDB) with each modification.

1. Контент или запросы в редакторах или окнах с установленными caching checkboxes будет сохраняться в памяти браузера при любых изменениях
2. Если caching checkboxes не установлены, отредактированный контент или новые запросы <b>НЕ СОХРАНЯТСЯ !!!</b> при перезагрузке страницы в браузере.
    Всегда обращайте внимание на оранжевый треугольник чтобы понять будет ли соответствующий контент сохранён или потерян при перезагрузке браузера

3. При установке caching checkboxes в любом редакторе или окне запросов создаётся новый workspace на основании загруженного ранее template.
    - Внизу в правой части footer всегда показано какой Template or Workspace загружен в данный момент
    - Окно  "Templates and Workspaces" позволяет загружать Workspaces, а также управлять Workspaces, то есть
    - редактировать имя или описание Workspace,
    - удалять Workspace,
    - а также сохранять Workspace в файловую систему или загружать из неё
    - Дополнительно для каждого Workspace указано: На основании какого Template он был создан и Какие компоненты кэшируются
4. Окно  "Templates and Workspaces" также позволяет загружать Templates, То есть примеры работы с Vason
