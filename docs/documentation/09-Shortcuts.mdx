# Shortcuts

**Alt+1** - focus on the `left editor`

**Alt+2** - focus on the `right editor`

**Alt+Q** - open the `query` window to analyze JSON in the left panel

**Alt+X** - open the `Xpath/XSLT` window to analyze JSON in the left panel

**Alt+F** - open the `Filter` window to analyze JSON in the left panel

**Alt+T** - open the `HTTP` window (`HTTP requests` tab)

**Alt+V** - open the `HTTP` window (`HTTP variables` tab)

**Alt+C** - open the `HTTP` window (`HTTP script` tab)

**Alt+S** - open the `Notes` window (this window serves as a clipboard or scratches texts)

To quickly access the previous window, just repeat the same shortcut.

**Ctrl+S** - save content in the browser cache and create a workspace automatically.

Shortcuts useful in panel editors, `HTTP`, `query` or `Notes` windows:

**Ctrl+D** - duplicate text in editors

**Ctrl+Y** - delete line in editors

**Ctrl+B** - go to brackets in editors

**Ctrl+Shift+B** - select to brackets in editors
