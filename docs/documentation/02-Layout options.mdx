# Layout options

Рабочая область приложение состоит из двух панелей с редакторами Json контента

import ThemedImage from '@theme/ThemedImage';

<ThemedImage
alt="Layout options"
sources={{
light: '/img/Layout_options_light.png',
dark: '/img/Layout_options_dark.png',
}}
/>

Так же в рабочей области содержатся следующие элементы:

1. Кнопки скрытия панелей
2. Общие элементы управления
3. Кнопка открытия окна управления templates и workspaces; Кнопка переключения тёмной и светлой тем
4. Элементы управления правой или левой панелей
5. Область управления открытыми окнами, в которой можно активировать или закрыть открытые окна
6. checkbox кэширования содержимого редакторов правой или левой панели
7. Элементы управления общие для всех редакторов
8. Панель навигации по тексту breadcrumbs

Клавиша "F1", нажатая в любом редакторе, вызывает список доступных команд и shortcuts к ним
